# AI Resume Screener

A web application that automatically screens resumes and calculates a match percentage against a job description using AI and NLP.
---

## Features

- Upload resumes in PDF format
- Paste job description to compare
- Calculates similarity score using TF-IDF & Cosine Similarity
- User-friendly web interface built with Flask

---

## Technologies Used

- **Python** - backend logic
- **Flask** - web framework
- **PyPDF2** - extract text from PDFs
- **scikit-learn** - TF-IDF & similarity calculation
- **HTML/CSS/Bootstrap** - frontend
